Jquery plugin to restrict and format input number

Installation and Usage

You have just to include the javascript file

<script src="path/input-number-format.jquery.min.js"></script>

Usage

$('.input-float').inputNumberFormat();

Example

Basic Float

/* HTML */ <input class="input-float" type="text" value="" />
/* JS   */ $('.input-float').inputNumberFormat();

Custom decimal precision with data

/* HTML */ <input class="input-float-4" data-decimal="4" type="text" value="" />
/* JS   */ $('.input-float-4').inputNumberFormat({ 'decimal_auto': 4 });

Integer

/* HTML */ <input class="input-integer" type="text" value="" />
/* JS   */ $('.input-integer').inputNumberFormat({ 'decimal': 0 });

Options

Option name Type Default value Description
decimal Integer 2 Number of authorized decimal(s)
decimal_auto Integer 2 Number of decimal(s) to add automatically on blur event
decimal_separator String "." Seperator to use
decimal_separator_authorized Array of string [".", ","] Seperator(s) authorized
